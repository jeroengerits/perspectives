# Ideas

Use this skill to expand the option space.

## When To Use

Use this skill for:

- generating genuinely different options
- reframing the problem to unlock new paths
- combining ideas into stronger hybrids
- turning risks or constraints into design prompts

## When NOT To Use

Do not use this skill for:

- repeating the current idea with cosmetic changes
- ignoring hard constraints in the name of creativity
- collapsing into a final recommendation
- replacing factual or risk analysis with brainstorming

## Purpose

Ideas prevents premature convergence by creating useful alternatives before the decision is closed.

## Collaboration Contract

When used inside `jeroengerits-perspective`:

- this skill is a required feedback input
- generate multiple useful options before narrowing
- build on what `jeroengerits-facts`, `jeroengerits-feelings`, `jeroengerits-benefits`, and `jeroengerits-risks` surfaced
- stay aware of hard constraints while exploring

When used alone:

- expand the option space without claiming the final answer

## Required Feedback

When called by `jeroengerits-perspective`, return:

- `Alternative Options`
- `Reframes`
- `Hybrids And Combinations`
- `Mitigations`
- `Most Promising New Angle`

## Focus

Prioritize:

- alternative approaches
- reframed problem statements
- hybrid options
- constraint-aware creativity
- risk mitigations
- simplifications
- unconventional but plausible moves

## Rules

- Generate multiple options before narrowing.
- Prefer useful novelty over novelty for its own sake.
- Stay aware of hard constraints even while exploring.

## Do Not

- repeat the current option with trivial wording changes
- propose ideas that ignore the actual objective
- drift into final recommendation mode

## Output Format

- `Alternative Options`
- `Reframes`
- `Hybrids And Combinations`
- `Mitigations`
- `Most Promising New Angle`
